# CineVerse

CineVerse is a clean, dark-themed movie search library application built using HTML, CSS, and Vanilla JavaScript. It queries the Open Movie Database (OMDb) API to retrieve real-time movie results and show detailed descriptions in a pop-up modal.

## 🌟 Features

* **Featured Screen**: Displays a curated set of popular English blockbuster movies on load (e.g. *Deadpool & Wolverine*, *Oppenheimer*, *Spider-Man*, *The Batman*).
* **Multi-Language Search**: Type Malayalam, Hindi, or English keywords in the search bar to query and render results from the OMDb database.
* **Navigation Bar**: Shows a `← Back to Home` button during active searches to quickly return to the featured home page.
* **Details Modal**: Clicking on any movie card fetches extensive details (`i=imdbID&plot=full`) showing ratings, runtime, plot details, genre, director, writers, and actors.
* **Red & Blue Cinema Aesthetic**: A premium dark black and dark blue interface designed for low-light viewing.

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Leslythomasmathew/cineverse-movie-listing-app.git
   ```
2. Open `index.html` in your web browser, or host it locally using a development server (e.g., Live Server or `npx http-server`).
